ブログエディタ 操作マニュアル
グリッドレイアウトまたはカードレイアウトでブログ記事を視覚的に作成し、スタンドアロンHTMLとしてエクスポートできるエディタです。

画面構成
アプリは ヘッダー、タブナビゲーション、タブコンテンツ の3層で構成されています。

ヘッダー: アプリ名と「📄 HTMLエクスポート」ボタン
タブ: 「⚙ 設定」と「✏ エディタ」の2つ

起動すると「設定」タブが表示されます。まず設定を済ませてからエディタに進むのが基本の流れです。

1. 設定タブ
1-1. レイアウトモードの選択
エディタの基本レイアウトを2種類から選びます。
モード説明グリッド（3列）3列 × 36行 = 108セルの自由配置。テキスト・画像を好きなセルに配置カード正方形カード（画像エリア60% + テキストエリア40%）を並べるモード

トグルボタンで切り替えます。
カードモード を選ぶと「カード枚数」の入力欄が出現します（2〜48枚、デフォルト12枚）。
PC表示は2列、モバイル表示は自動で1列になります。

1-2. 画像ディレクトリ設定（パスプリセット）
画像をセルにドラッグ＆ドロップした際に、自動でHTMLエクスポート用のパスを組み立てるための設定です。
プリセットの追加手順：

「エイリアス名」に短い識別名を入力（例: post-img）
「ディレクトリパス」にパスを入力（例: /images/posts/）
「追加」ボタンをクリック

初期状態では以下の3つが登録されています。
エイリアスパスpost-img/images/posts/bg/images/bg/icons/images/icons/
登録済みプリセットは一覧表示され、各行の「✕」ボタンで削除できます。

注意: パスの末尾には必ず / を付けてください。相対パス・ルート相対パス・URLのいずれも使用可能です。

1-3. デフォルトパスの選択
登録したプリセットの中から、エディタで画像をドロップした際に使用するディレクトリを選びます。プルダウンメニューから選択するだけで設定完了です。
1-4. 背景画像
エクスポートされるHTMLの全画面固定背景画像を指定します。

「背景画像パス」に画像のパスを入力（例: /images/bg/hero.jpg）
空欄のままにすると背景画像なしでエクスポートされます

1-5. ページ背景色
プレビューとエクスポートの両方に適用される背景色を設定します。

カラーピッカーで色を選択
選択中の色はHEXコードで表示されます
「リセット」ボタンで初期値 #f5f5f5（薄いグレー）に戻ります


ヒント: 暗い背景色を設定すると、グロー（発光）エフェクトがより映えます。

1-6. エクスポート設定

ページタイトル: エクスポートHTMLの <title> タグに反映されます
CSS読込パス: 外部CSSファイルのパスを指定すると、エクスポートHTMLに <link> タグが追加されます。空欄ならインラインCSSのみ

自動保存について
設定タブで行ったすべての変更は localStorage に自動保存されます。保存時には画面下部に「✓ 自動保存済み」と一時表示されます。ブラウザを閉じて再度開いても設定は維持されます。

2. エディタタブ
「✏ エディタ」タブをクリックすると、左に サイドパネル、右に プレビューエリア が表示されます。
2-1. プレビューエリアの基本操作
プレビューエリアには設定で選んだレイアウトモードに応じてグリッドまたはカードが表示されます。

各セル（またはカード）の左上に番号バッジが表示されます

グリッド: 行番号-列番号（例: 1-1, 1-2, 1-3）
カード: 通し番号（例: 1, 2, 3...）


セルをクリック すると選択状態になり、赤い枠線で囲まれます
プレビュー内のセル以外の領域をクリックすると 選択解除 されます

2-2. 画像のドラッグ＆ドロップ
セルに画像を配置する手順：

パソコンのフォルダからエディタのセル上に画像ファイルをドラッグ
セルの上にドラッグすると緑色のドロップゾーンオーバーレイが出現
ドロップすると画像がセルに配置され、プレビューに表示されます

画像をドロップすると、自動的に以下が設定されます：

プレビュー: FileReaderで読み込んだDataURLが表示
エクスポートパス: 設定のデフォルトパス + ファイル名（例: /images/posts/photo.jpg）

サイドパネルの画像情報エリアには、サムネイル・ファイル名・エクスポートパスが表示されます。
画像を削除するには：

グリッドモード: セルにマウスを載せると右上に表示される赤い「✕」ボタンをクリック
サイドパネル: 画像情報横の「✕」ボタンをクリック


注意: サイドパネル上部の「画像ドロップ先」表示で、現在どのディレクトリが使われるか常に確認できます。


3. セル編集パネル（サイドパネル）
セルを選択すると、サイドパネルに以下の編集項目が表示されます。
3-1. テキスト入力
テキストエリアに自由にテキストを入力できます。入力した内容はリアルタイムでプレビューに反映されます。
3-2. 入力モード
モード説明プレーン入力したテキストがそのまま表示されます（改行もそのまま反映）マークダウンMarkdown記法で書式付きテキストを入力できます
マークダウンモードで使える記法：
記法結果# 見出し大見出し（h1〜h4まで # の数で指定）**太字**太字*斜体*斜体~~取り消し線~~取り消し線`コード`インラインコード``` で囲むコードブロック- 項目箇条書きリスト1. 項目番号付きリスト> 引用引用ブロック---水平線[テキスト](URL)リンク![代替テキスト](URL)画像埋め込み
独自拡張記法：
記法結果{テキスト|1.5}テキストを1.5倍のサイズで表示{テキスト|0.8}テキストを0.8倍（80%）のサイズで表示{漢字|かんじ}ルビ（ふりがな）を表示

サイズ記法とルビ記法の違い: | の後の値が数値のみならサイズ変更、数値以外を含む場合はルビとして処理されます。

3-3. テキストサイズ変更ツール（マークダウンモード限定）
マークダウンモードを選択すると、テキストサイズ変更ツールが表示されます。
使い方：

テキストエリア内でサイズを変更したい部分を マウスで選択
ボタンをクリック：

「🔍 小」: 選択テキストを {テキスト|0.8} で囲む（80%サイズ）
「↺ 標準」: {テキスト|1.0} で囲む（標準サイズ）
「🔍 大」: {テキスト|1.2} で囲む（120%サイズ）
「🔍 特大」: {テキスト|1.5} で囲む（150%サイズ）



3-4. 文字方向
設定説明横書き通常の横書きテキスト（左から右）縦書き縦書きテキスト（右から左、上から下）
3-5. 配置
横揃え（水平方向）： 左 / 中央 / 右
縦揃え（垂直方向）： 上 / 中央（デフォルト） / 下

縦書きモード時: 横揃えと縦揃えの方向が入れ替わります（横揃えが上下方向、縦揃えが左右方向に対応）。

3-6. フォント設定
項目選択肢書体ゴシック体（Noto Sans JP）/ 明朝体（Noto Serif JP）サイズ8px〜120px（デフォルト: 16px）太さLight (300) / Regular (400) / Medium (500) / Bold (700)文字色カラーピッカーで自由に指定（デフォルト: #333333）
3-7. 装飾
背景色と不透明度：

カラーピッカーで背景色を選択
「クリア」ボタンで透明に戻す
スライダーで不透明度を 0%〜100% で調整

角丸（border-radius）： スライダーで 0px〜50px（デフォルト: 10px）
影（box-shadow）：
種類効果なし影なし小さい影控えめなドロップシャドウ中くらいの影標準的なドロップシャドウ大きい影存在感のあるドロップシャドウグロー（発光）ネオン風の発光エフェクト（色を指定可能）ハードシャドウフラットデザイン風の硬い影
「グロー（発光）」を選ぶと 発光色 のカラーピッカーが追加表示されます（デフォルト: #e94560）。
アニメーション：
種類効果なしアニメーションなしフェードイン透明から徐々に表示スライドアップ下から上にスライドしながら表示スライド（左から）左からスライドして表示スライド（右から）右からスライドして表示ズームイン小さい状態から拡大して表示バウンス弾むように表示パルス（繰り返し）拡大・縮小を繰り返すふわふわ（繰り返し）上下にふわふわ動き続けるクリックでフェードアウトクリックするとフェードアウト（再クリックで復帰）
アニメーションを設定後、「▶ プレビュー」ボタンでエディタ内で動作確認できます。

4. HTMLエクスポート
ヘッダーの「📄 HTMLエクスポート」ボタンをクリックすると、現在の編集内容がスタンドアロンHTMLファイルとしてダウンロードされます。
エクスポートの仕組み

グリッドモード: テキストまたは画像が含まれるセルだけがエクスポートされます。空の行は自動的にスキップ
カードモード: テキストまたは画像が含まれるカードだけがエクスポート
ファイル名: 設定タブの「ページタイトル」をベースに生成（空白はハイフンに変換）

エクスポートHTMLの特徴

Google Fonts（Noto Sans JP / Noto Serif JP）の読み込みを含む
設定した背景色・背景画像が反映
すべてのアニメーションCSSが含まれる
レスポンシブ対応: 768px以下でグリッドが1列に自動変換
「クリックでフェードアウト」アニメーションのJavaScriptも含まれる
外部CSSパスを設定していた場合は <link> タグも含まれる

エクスポート前のチェックリスト

設定タブの「ページタイトル」を入力したか
画像のエクスポートパスが正しいか（サイドパネルの画像情報で確認）
背景色・背景画像の設定は意図通りか
外部CSSを使う場合、パスは正しいか


重要: エクスポートされるHTMLには画像の実データ（DataURL）は含まれません。エクスポートパスで指定した場所に実際の画像ファイルを配置する必要があります。


5. レスポンシブ対応
エディタ画面
画面幅レイアウト変化901px以上サイドパネル（左）+ プレビュー（右）の横並び900px以下サイドパネル（上、最大300px）+ プレビュー（下）の縦並び768px以下グリッドが1列に変換
エクスポートHTML
画面幅レイアウト変化769px以上グリッド: 3列表示 / カード: 2列表示768px以下両モードとも1列表示に自動変換

6. キーボード操作・ショートカット
このエディタには専用のキーボードショートカットはありません。基本的にマウス操作とサイドパネルのUI部品で操作します。テキスト入力時は通常のテキストエリアと同じキーボード操作が使えます。

7. データの保存について

設定データ: localStorage に自動保存（ブラウザを閉じても維持）
セルの編集内容: 保存されません。ページをリロードするとセルのテキスト・画像・装飾はすべてリセットされます


重要: 作業内容を残したい場合は、必ずHTMLエクスポートを行ってからページを離れてください。


8. トラブルシューティング
問題原因・対処画像をドロップしても反映されない画像ファイル以外（PDFなど）をドロップしていないか確認。画像形式（jpg, png, gif, webp等）のみ対応エクスポートHTMLで画像が表示されないエクスポートパスの場所に実際の画像ファイルを配置してください設定が反映されないブラウザのlocalStorageが無効になっていないか確認レイアウトが崩れるブラウザの横幅を広げてみてください。900px以上推奨グローエフェクトが見えにくいページ背景色を暗い色に変更してくださいマークダウンが正しく表示されない記法の前後にスペースや改行が必要な場合があります（例: # の後にスペース）テキストサイズ変更ボタンが表示されない入力モードを「マークダウン」に切り替えてください
